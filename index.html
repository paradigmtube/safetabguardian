Privacy Policy

Effective Date: June 3, 2025

SafeTab Guardian does not collect, transmit, or share any personal data.

What We Store

The extension uses Chrome's storage API to locally store the following user preferences:

  • Toggle state (blocking on/off)
  • Custom whitelist domains
  • Blocked attempt counter

This data is stored either locally or using Chrome Sync and is not accessible to us or any third party.

What We Do NOT Collect

  • Personally identifiable information (PII)
  • Browsing history or page content
  • Location or IP address
  • Emails, messages, or user activity
  • Passwords or authentication data

Third-Party Services

This extension does not use analytics, tracking scripts, or load code from any remote servers.
